## Topics
- **[Purpose-Built AI Services](0_Purpose_Built_AI_Services/)** - Learn about AWS's purpose-built AI services including Transcribe, Polly, Translate, Comprehend, Rekognition, Textract, and Lex.
- **[Multilingual Chat Demo App](1_Purpose_Built_AI_Services_Demo_Apps/Multilingual_Chat/)** - Real-time WebSocket-based chat application with automatic translation and sentiment analysis, allowing three users to chat in English, Spanish, and French with messages translated in real-time.
- **[Insurance Claim Submission Demo App](1_Purpose_Built_AI_Services_Demo_Apps/Insurance_Claim_Submission/)** - Full-stack application demonstrating AWS AI services for processing insurance claims with multi-language support, document processing, and visual analysis.
- **[Deploy a Full-stack App with AI Services Integration to AWS](1_Purpose_Built_AI_Services_Demo_Apps/AWS_App_Deployment/)** - Deploy a full-stack claim processing application to AWS using CDK, including frontend, Lambda functions, API Gateway, S3, and DynamoDB and AI Services integration.
- **[Amazon Bedrock](2_Amazon_Bedrock/)** - Chat application using AWS Bedrock with a FastAPI backend and Streamlit frontend; configurable models, streaming, and conversation history.
- **[Bedrock App Integration](3_Bedrock_App_Integration/)** - Full-stack automobile damage assessment: submit a description and three photos (wide-angle, straight-on, closeup) for an AI assessment, with a React chat UI and admin dashboard.
- **[Bedrock RAG with Knowledge Bases](4_Bedrock_RAG_KB/)** - Retrieval-Augmented Generation (RAG) demo: ask questions over your documents using a Bedrock Knowledge Base, with a FastAPI backend and optional Streamlit or React frontend.
